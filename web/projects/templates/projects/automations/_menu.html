{% if automation.trigger_type == 'report' %}

Report automations collect matching CVEs over a full daily or weekly period, based on the selected timezone.

Actions are executed later, at the scheduled time, on the CVEs collected in that report. For example, a daily report sent at 09:00 includes CVEs from the previous full day.

{% else %}

Alert automations run every hour on CVEs that matched during the previous hour.

Use them for near real-time actions, such as sending a notification as soon as a CVE enters the project or matches your conditions.

{% endif %}
