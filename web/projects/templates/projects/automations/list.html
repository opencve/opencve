Automations define how OpenCVE reacts to CVEs that match your project subscriptions.

Use conditions to decide which CVEs should match, then choose actions such as sending a notification, assign a user, or updating the CVE status.

Alert automations run as soon as matching CVEs are detected. Report automations collect matching CVEs over a daily or weekly period, then run their actions at the scheduled time.
